# FakeBook - A C++ Facebook Clone

FakeBook is a comprehensive Facebook-inspired social networking application developed in C++. The project demonstrates the evolution of a software system from a console-based implementation to a modern graphical desktop application while following object-oriented design principles and software engineering best practices.

The project is divided into two development phases:

## Phase 1 - Console Application
The first phase focuses on implementing the complete backend and core functionality using C++ and object-oriented programming concepts.

Features include:
- User Authentication
- User Registration & Login
- Strong Input Validation
- Friend Requests
- Friend Suggestions
- Mutual Friends
- Public & Private Profiles
- News Feed
- Posts
- Likes & Comments
- Notifications
- User Search
- File Persistence
- Linked List Based Data Structures

This phase emphasizes algorithms, data structures, OOP concepts, memory management, and backend logic.

---

## Phase 2 - Desktop GUI Application
The second phase transforms the console application into a modern desktop application with a professional graphical interface.

Features include:
- Modern Desktop UI
- Interactive News Feed
- Profile Management
- Friend Management
- Notifications
- Search Interface
- Settings
- Real-Time UI Updates
- Improved User Experience
- Modular GUI Architecture

The backend remains independent from the user interface, demonstrating clean software architecture and separation of concerns.
